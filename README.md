# 🏪 STORE RATING SYSTEM

> Empowering Retailers, Enhancing User Experience, and Streamlining Feedback with Real-Time Insights

---

### 🚀 Overview
The **Store Rating System** is a full-stack web application designed to allow users to rate and review stores while store owners can monitor feedback analytics.  
Built using **React** for the frontend and **Node.js (Express)** for the backend, it ensures seamless communication between client and server with secure authentication.

### 📋 Features
- 🔐 **JWT Authentication** – Secure login and role-based access for users and store owners  
- ⭐ **Store Rating System** – Users can rate stores and leave feedback  
- 🧑‍💼 **Store Owner Dashboard** – Manage ratings and customer responses  
- 📊 **Real-Time Data Display** – Ratings and analytics update dynamically  
- ⚙️ **Password Change Feature** – Built-in security for users and store owners  
- 💾 **Persistent Storage** – All data securely stored in MySQL database
